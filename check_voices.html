需要安装英语语音包

Windows 11 安装步骤：

  1. 打开 设置 → 时间和语言 → 语言和区域
  2. 点击 添加语言，搜索 English
  3. 选择 English (United States) 或 English (United Kingdom)
  4. 安装后点该语言 → 语言选项 → 下载 语音
  5. 重启浏览器后回来再测试

或者：设置 → 时间和语言 → 语音 → 管理语音 → 添加语音 → English
