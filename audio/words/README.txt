Put word pronunciation MP3 files here.

Expected paths:
- audio/words/uk/about.mp3
- audio/words/us/about.mp3

File names must be lowercase English words. For words with hyphens, keep the hyphen.
The web app plays these files first and falls back to browser TTS when a file is missing.
The same directory structure can be reused in a WeChat Mini Program with wx.createInnerAudioContext().
